# Structure du Projet

## Fichiers Principaux

**environment.py** : Fonctions d'initialisation de l'environnement et intégration des méthodes de politique, d'entraînement et d'évaluation.

**baseline.py** : Méthodes de base pour la définition heuristique de stratégies.

**utils.py** : Fonctions auxiliaires pour les méthodes baseline.

**IQL** : Classes et implémentations spécifiques à la méthode Independent Q-Learning (IQL).

**QMix** : Classes et implémentations spécifiques à la méthode QMix.

**visualization** : Fonctions de visualisation pour les trois méthodes (baseline, IQL, QMix).

**io_utils** : Utilitaires de sauvegarde/chargement de modèles et conversion en politique.

## Documentation et Résultats

**Report.ipynb** : Rapport complet et résultats expérimentaux.

**medium_test** : Résultats des tests générés par Report.ipynb.
